# SiteEscola

SiteEscola is a simple static landing page themed around space exploration. The project is built with plain HTML and CSS, using local image assets and the Roboto font from Google Fonts.

## Preview

The current page includes:

- A header with the project logo and navigation links.
- A full-screen hero section with a universe-themed background image.
- A main call to action: **"Quero saber mais"**.

## Getting Started

No package manager or build step is required. To view the project locally, open `index.html` directly in a browser.

### Option 1: Open the file directly

```bash
xdg-open index.html
```

### Option 2: Run a local development server

If you have Python installed, run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Customization

- Update page content in `index.html`.
- Change layout and visual styles in `styles/index.css`.
- Replace or add images in the `assets/` directory.

## Notes

This project is currently a front-end static page. Navigation links and the call-to-action link are placeholders and can be connected to additional pages or sections later.
